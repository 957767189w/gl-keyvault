gl-keyvault v0.1.0
Secure API key management proxy for GenLayer Intelligent Contracts.

Endpoints:
  POST /api/proxy          - Proxy authenticated API requests
  POST /api/keys/register  - Register a new key alias
  GET  /api/keys/list      - List registered aliases
  POST /api/keys/rotate    - Rotate an existing key
  GET  /api/keys/audit     - Query usage audit log
  GET  /api/health         - Service health check

Docs: https://github.com/genlayer-foundation/gl-keyvault
